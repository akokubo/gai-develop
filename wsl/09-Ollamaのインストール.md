# Ollamaのインストール

Ubuntuで以下のように打ってインストールする

```sh
curl -fsSL https://ollama.com/install.sh | sh
```

# Ollamaのアップデート

インストールと同様だが、Ubuntuで以下のように打ってアップデートする

```sh
curl -fsSL https://ollama.com/install.sh | sh
```
